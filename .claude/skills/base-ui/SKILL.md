# Base UI Skill

When working with `@base-ui/react` components, use this skill to fetch accurate documentation.

Base UI is a composable, styling-agnostic component library for React.

## How to use

1. Reference `llms-index.md` to find the correct documentation URL for the component
2. Use WebFetch to retrieve the specific component documentation from base-ui.com
3. Follow Base UI's composable, styling-agnostic patterns

## Key resources

- Quick start: https://base-ui.com/react/overview/quick-start.md
- Styling guide: https://base-ui.com/react/handbook/styling.md
- Animation: https://base-ui.com/react/handbook/animation.md
- Forms: https://base-ui.com/react/handbook/forms.md
- Full component index: See llms-index.md

## When to activate

Activate this skill when:
- User asks to create/use Base UI components
- User asks how to style or customize Base UI
- User references @base-ui/react package
- Working with any component listed in llms-index.md
